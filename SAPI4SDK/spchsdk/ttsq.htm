Text-to-Speech Queue Sample

The TTS Queue Sample demonstrates how to use the Text-to-Speech Queue Object. (See the TTS Queue documentation in the Speech Tools section.) You can use the queue to play two or more text-to-speech voices sequentially, and to intermix wave files. The demo's functionality has been kept simple so it's easier to understand the code.

To explore the functionality of the TTS queue object, try the following:

  1. Run the Text-to-speech queue sample.
  2. Press "Pause" so you have enough time to type in text.
  3. Make sure that the "Male Voice" is selected.
  4. Type some text into the edit box, such as "This is the male voice."
  5. Press "Speak." You won't hear anything yet because the audio is paused.
  6. Click on the "Female Voice" radio button to choose it.
  7. Type in some text, such as "This is the female voice."
  8. Press "Speak."  You won't hear anything yet because the audio is paused.
  9. Press "Resume" to have the voices speak in sequence.
  10. You can also experiment with having sounds played.
  11. Check the "Play beep after every buffer."
  12. Type some text into the edit box.
  13. Press "Speak", and it will speak the text, followed by a beep.
  14. To add your own audio, press "Add Wave".
  15. In the dialog box's first edit field, type the full path to a 22 kHz 16-bit mono wave file. If you don't have one, you can use the wave editor to record such a file. (The wave editor defaults to a 16 kHz 16-bit mono wave file, so you'll need to change the data format.)
  16. In the second edit field, type in a string that will activate the beep, such as "MySound".
  17. Press OK to load the wave file in.
  18. In the demo's main window, type in "MySound" by itself. Make sure there's no other text or surrounding space.
  19. Press "Speak." Your wave file will be played. It doesn't matter which voice you choose.
  20. If you want to queue up text to speak along with your wave file, then:
  21. Press "Pause".
  22. Type in any text, except "My Sound".
  23. Press "Speak." You won't hear anything yet because the audio is paused.
  24. Type in "MySound".
  25. Press "Speak." You won't hear anything yet because the audio is paused.
  26. Type in some different text.
  27. Press "Speak." You won't hear anything yet because the audio is paused.
  28. Press "Resume". The voice will speak and then your sound will be played, and then it will speak again. If you have the "Play beep after every buffer" turned on then you'll also hear three beeps.
