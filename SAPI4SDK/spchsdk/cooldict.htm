Microsoft Dictation

If you're looking for a dictation demo to try, then run Microsoft Dictation. Microsoft Dictation will already be installed if you have the Speech SDK Suite.

  • Run it!

The "Run it!" link only works if Microsoft Dictation was installed under c:\program files\Microsoft dictation. If you installed under a different directory, you can run Microsoft Dictation by clicking on the Start menu, looking under Programs, and selecting Microsoft Dictation.

 

If you didn't install the Speech SDK Suite, you should download the 4.0 Speech SDK Suite from the Internet.
