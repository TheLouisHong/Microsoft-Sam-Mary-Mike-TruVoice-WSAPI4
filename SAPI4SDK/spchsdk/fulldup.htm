Full Duplex Demo

The Full Duplex Demo tests to see if your sound card supports full duplex, and if it does, allows you to interrupt its speech.

To run the demo:

  1. You should either wear a close-talk headset and use speakers, or use headphones. Otherwise, when the text-to-speech engine speaks, the recognizer will hear the text-to-speech, and react to it. If your sound card supports "echo cancellation" you don't have to take this precaution, but very few, if any, sound cards support echo cancellation.
  2. Run it
  3. In the engine selection dialog, select the speech recognition and text-to-speech engines that you wish to use, along with the sound cards.
  4. Press OK to proceed. If there is any problem getting full duplex to work, a message box will be brought up. For full duplex to work (on most sound cards), the recognizer and text-to-speech engine must use the same sampling rate. The combination of some speech recognizers and text-to-speech engines may not work because they require different sampling rates.
  5. Another dialog, "Full duplex demo" will appear. Speak a number, such as "fifty two million". Text-to-speech will read back the number. While the computer is speaking, speak another number. The computer will stop reading the previous number and read back what you just spoke.
  6. That's it.

You can best take advantage of the demo by stepping through the code and seeing how the full duplex detection is done, and by using the code in your own application.
