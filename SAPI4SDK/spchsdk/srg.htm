Speech Recognition Grammar Compiler

The speech recognition grammar compiler lets you load, modify, and test grammars. It supports context-free grammars, limited domain grammars, and dictation grammars. To use the application, run it and open one of the grammar .txt files. Select Grammar\Compile to compile the grammar. If there are no errors then select Grammar\Test to test it.

For documentation on the grammar format, look in "Grammar Text File Format" under the "Speech Tools API" section of the COM documentation.

The source code shows you how to use speech recognition engines and the grammar compiler object. You can turn on the USELOGGING switch in the compiler to demonstrate how to use the Speech Tools logging object.

You need this to run:

  • Command & Control Speech Recognition Engine, or
  • Limited Domain Recognition Engine, or
  • Dictation Recognition Engine
