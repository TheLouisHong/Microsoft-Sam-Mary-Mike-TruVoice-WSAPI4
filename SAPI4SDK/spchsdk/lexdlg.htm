Lexicon Editor

The lexicon editor allows you to change the pronunciation of a word, modifying lexicons in both speech recognition and text-to-speech. The sample code demonstrates the use of the Lexicon Dialog, provided by Speech Tools, and how to use ILexPronounce2.

To run the sample:

  1. Run it.
  2. Select the speech recognition or text-to-speech engine whose lexicon you wish to modify. An engine will only appear on the list if it supports ILexPronounce.
  3. Press OK.
  4. If you select a dictation engine, the application may ask if you want to modify the dictation lexicon. If you modify the dictation lexicon, you can add and remove words from the dictation vocabulary, controlling what words will get recognized. If you answer no, you will be modifying the user lexicon for context-free-grammars. The CFG user lexicon usually starts out empty.
  5. The lexicon dialog will appear.
  6. You can add, remove, or modify any word by pressing "Edit another". This will ask you what word to modify and how you wish to modify it. It uses the Speech Tools lexicon dialog.
  7. To bulk-edit words, then press "Scan file for words".
  8. In the "Scan file" dialog, type in the name of a text file. The application will produce a list of all the words appearing in the text file. Using the radio buttons, the list can be pared down to only those words not in the lexicon, or only those words in the lexicon. (The radio buttons will be disabled if the engine only supports ILexPronounce, and not ILexPronounce2.) You can also only keep words appearing more than a specified number of times, so that uncommon words will be ignored.
  9. Press OK in the "Scan file" dialog to scan the document. It may take awhile to complete.
  10. The "Lexicon" dialog's "Words from scanned file" list box will be filled with the words from the file.
  11. Click on a word to hear what it sounds like, view its pronunciations, and Edit it.
  12. "Add all" will search through the words in the list, and add any words to the lexicon which are not already present. (This will be grayed out if the engine doesn't support ILexPronounce2.)
  13. "Remove all" will remove all the words in the list from the lexicon. Warning: Removing common words may degrade the performance of the engine. (This will be grayed out if the engine doesn't support ILexPronounce2.)

 

You need this to run:

  • Speech Recognition or Text-to-Speech Engine that supports the ILexPronounce interface.
