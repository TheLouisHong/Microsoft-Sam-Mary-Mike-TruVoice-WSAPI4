Record from Voice Commands/Dictation

Play to Voice Commands/Dictation

RecPlay.exe allows you to record speech being spoken into the speech recognizer, if the application is using Voice Commands or Voice Dictation for the recognition. It also allows you to route the audio from a wave file into the application using Voice Commands or Voice Dictation, rather than having the application listening to the PC microphone.

You can use RecPlay.exe to collect recordings of real users using your application. You can then use the recordings to analyze the accuracy of speech recognition in your application.

To record a session:

  1. Run an application that uses Voice Commands/Dictation. If you're just testing out this applet, you can try running Microsoft Voice or Microsoft Dictation.
  2. Run the RecPlay.exe application.
  3. Make sure the "Recording" tab is visible.
  4. Under "File to write audio to", type in a wave file, such as "c:\out.wav".
  5. Press "Start".
  6. Turn listening on in the application and speak. When you've recorded enough, turn listening off.
  7. In the RecPlay.exe application, press "Stop".
  8. To hear the recorded session, double click "c:\out.wav" in the Windows explorer.

Sometimes you may want to replay audio back into your application to see how it behaves. To replay the audio:

  1. Run an application that uses Voice Commands/Dictation. If you're just testing out this applet, you can try running Microsoft Voice or Microsoft Dictation.
  2. Run the RecPlay.exe application.
  3. Make sure the "Playback" tab is visible.
  4. Under "File to read audio from", type in a wave file, such as "c:\out.wav". The audio recording must be the same data format as the speech recognizer is using. If you used RecPlay.exe to record the audio, you shouldn't have any problem.
  5. Press "Start".
  6. Turn listening on in the application and wait. The audio will be passed into the application, even though you won't be able hear it. The application should react as though a person were speaking to it. When you've played all the audio, turn listening off.
  7. In the RecPlay.exe application, press "Stop".
