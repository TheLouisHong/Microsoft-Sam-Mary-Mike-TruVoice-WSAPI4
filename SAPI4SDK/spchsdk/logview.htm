Telephony Log Viewer

The telephony log viewer application allows you to view data logs written out by the STLog object. You can view the logs to learn how users interact with your telephony application.

You need this to run:

  • Nothing

You can:

  • Run it!
  • Show a sample log
  • View Sample Source Code
  • To run the log viewer, you can also double click a .stl file.
