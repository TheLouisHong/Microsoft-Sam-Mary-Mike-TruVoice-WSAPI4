Speak what you type

The Text-To-Speech Demo Application demonstrates simple text-to-speech by having the speech engine read text you enter.

Run the Text-To-Speech Demo Application (TTSAPP.EXE). Type text into the box and press the Speak button. The Text-To-Speech Demo reads back the text you entered. You can also select different voices and change speaking styles.
