Getting Started!

Welcome to the Microsoft Speech SDK! This page will tell you what you need for the Microsoft Speech SDK and sample applications to work.

The Microsoft Speech SDK works with Windows 95/98 and Windows NT 4.0 (or better) (x86, Alpha).

If you haven't already, you should check your microphone and record levels to make sure that speech recognition will actually hear what you're saying.

If you plan to do telephony application development, you should read this to make sure that your telephony card drivers are properly installed.

You need to make sure that you have a speech recognition and text-to-speech "engine" installed on your computer. The Microsoft Speech SDK may not have come with any engines so you may need to install them now. To use all of the samples in the SDK you'll need three different types of engines:

  • Command-and-control speech recognition - This allows users to give simple commands to their computer.
  • Dictation speech recognition - Can be used to transcribe what the user is speaking.
  • Text-to-speech - Reads out text.

If you installed the Microsoft Speech Suite or from the Microsoft Developers Network Level II, you should already have some engines installed. However, if you don't, you can contact an ISV that produces speech recognition or text-to-speech engines. Click on "Internet" to get to the engine vendors web pages.

Once you have an engine, try playing around.
