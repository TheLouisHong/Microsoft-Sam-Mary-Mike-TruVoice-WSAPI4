Voice Command Sample App

The Voice Command Demo demonstrates simple speech recognition by showing you the commands it recognizes.

Run VCMDDEMO. It cycles through a list of voice commands that the engine recognizes. Say any of these commands and VCMD Demo displays what it recognizes. If it does not recognize the command, it returns Unrecognized. You can add commands by clicking the Change Commands button and adding a command to the list; to delete a command, highlight it and press the DEL key.

You need this to run:

  • Command & Control Speech Recognition Engine
  • A navigator application must be running. You can install Microsoft Voice, or just run Feedback.
