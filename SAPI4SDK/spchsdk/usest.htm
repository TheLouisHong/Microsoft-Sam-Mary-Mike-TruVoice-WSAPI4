Speech Tools Sample Apps

This is a list of the sample applications using Speech Tools.

 

Samples new to 4.0 are highlighted in yellow.

 

Accuracy Test Allows you to test the accuracy of your speech recognizer and grammar.

 
Lexicon Editor Allows you to edit the lexicon of a speech recognition or text-to-speech engine.

 
Microphone Setup Wizard Demo how to use the microphone setup wizard.

 
Microsoft Dictation An application you can dictate into.

 
Microsoft Voice Use Voice Commands to control PC applications.

 
Phone Converter Demo Demonstrate converting phoneme strings to/from IPA.

 
Speech Recognition Functionality Test Tests how compatible the engine is with SAPI.

 
Speech Recognition Grammar Compiler Grammar compiler.

 
Speech Recognition Stress Test Tests the stability of a speech recognition engine under stressful conditions, such as long usage, multiple instances, or low resources.

 
Speech Wrapper Library Source code for the C++ speech wrapper.

 
Telephony Control Editor Load, edit, and test telephony controls.

 
Telephony Control Sample Sample telephony control code.

 
Text-to-Speech Queue Sample Queues up male and female voices, and wave files.

 
Wave Editor Allows you to edit wave files.
