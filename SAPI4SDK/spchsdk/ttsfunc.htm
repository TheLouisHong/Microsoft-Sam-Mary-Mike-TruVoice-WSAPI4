Text-to-Speech Functionality Tester

TTSFunc.exe is a functionality test application for text-to-speech engines. Its purpose is to make sure the engine complies with SAPI. A secondary purpose is to stress out the engine and find crashes and memory leaks.

Application developers that intend to bundle or recommend an engine, may wish to use the functionality test to insure the stability of the engine. Developers producing engines can use it as a test to make sure their engine is SAPI compliant.

To use the functionality test:

  • Click Run It!
  • Under Options, select the speech engines you wish to test. The speech recognition engine is used to determine if the text-to-speech engine spoke the right words.
  • Under Options, select the text-to-speech engine you wish to test.
  • If you wish to log the results, select "Logging" under "Options" and type in the file name.
  • Under Test, select the test cases you wish to run.
  • Let it run through the course of tests. At the end you'll be given a list of which tests passed/failed.

You need this to run:

  • Speech Recognition Engine
  • Text-to-speech engine
