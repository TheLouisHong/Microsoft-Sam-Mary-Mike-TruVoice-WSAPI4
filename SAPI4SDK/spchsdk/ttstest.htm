Direct Text-to-speech Hand Tester

TTSTest is an MFC dialog-based application for testing the text-to-speech engine interfaces. TTSTest allows you to perform comprehensive tests of the text-to-speech API and engine. TTSTest allows the following interfaces to be interactively exercised: ITTSCentral, ITTSAttributes, and ITTSDialogs. The ITTSEnum and ITTSFind interfaces are exercised when selecting an engine mode, and the ITTSNotifySink and ITTSBufNotifySink are provided by the application to report notifications sent by the engine. Controls for the optional ILexPronounce interface are provided, but are not supported by the test application.

Usage

The TTSTest application can be used to demonstrate a variety of engine features and perform ad hoc testing of the supported interfaces. Upon startup, you are presented with the main dialog containing the following components:

Add Mode button

Allows you to choose a single text-to-speech mode from the list of installed engines. Only one engine can be loaded at a time, however you can run multiple instances of the application to compare different engines.

Selected Engine Mode list box

The Selected Engine Mode list box displays the engine currently selected (via the Add Mode button). Exit the application to select a different mode.

ITTSCentral::TextData

This button is a "global" duplicate of the TextData button on the ITTSCentral property page. This invokes the function that will convert text data to speech, and is placed here as a convenience. For instance, you may want to watch notifications on the "Notifications" tab while speaking a phrase entered on the ITTSCentral tab.

Interface properties

These property pages (tabbed dialogs) provide the user interface for exercising the various interfaces (once an engine is selected). The tab name corresponds to the available TTS interfaces.

Main output window

This is the bottom output window which contains general error information and status.

Typical usage instructions:

  1. Run TTSTest.
  2. Press Add Mode to bring up a dialog containing available engine modes.
  3. Select a mode and return to the main dialog.
  4. Type in a phrase to say in the edit control on the ITTSCentral tab.
  5. Click either the ITTSCentral::TextData button on the main dialog, or the TextData button on the ITTSCentral tab to say the phrase.

Note

One or more text-to-speech engines must be properly installed on the system running this application to enable the user to successfully select an engine.
