Phoneme Demo

The PConv.exe application demonstrates the Phoneme Converter object in the speech tools library. It allows you to convert between IPA and human-readable phonemes using the ISTPhoneConv interface. For a description of the human-readable phonemes, see our documentation.

You need this to run:

  • Windows NT - The phoneme converter object works on Windows 95/98, but this demo uses a unicode font to show the IPA characters.

You can:

  • Run it! (Demo uses Unicode, so it only works on Windows NT)
