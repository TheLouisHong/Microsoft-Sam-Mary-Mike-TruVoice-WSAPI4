Cool Command & Control Demo

If you're looking for a cool Command & Control speech recognition demo to try, then run Microsoft Voice. Microsoft Voice will already be installed if you have the Speech SDK Suite.

  • Run it!

The "Run it!" link only works if Microsoft Voice was installed under c:\program files\Microsoft voice. If you installed under a different directory, you can run Microsoft Voice by clicking on the Start menu, looking under Programs, and selecting Microsoft Voice.

 

If you didn't install the Speech SDK Suite, you should download the 4.0 Speech SDK Suite from the Internet.
