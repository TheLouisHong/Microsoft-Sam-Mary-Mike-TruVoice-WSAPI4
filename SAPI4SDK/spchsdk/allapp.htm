All Sample applications

This is a complete list of the sample applications provided in the Microsoft Speech SDK.

 

Samples new to 4.0 are highlighted in yellow.

 

Accuracy Test Allows you to test the accuracy of your speech recognizer and grammar.

 
Answering Machine Sample Sample answering machine using telephony controls.

 
Audio Destination File Writes the text-to-speech audio to a wave file.

 
Audio Collection Tool Tool that easily allows you to record speech samples for accuracy tests.

 
Audio Indexer The audio indexer demonstrates how to use a speech recognition engine to generate a keyword index from wave files.

 
Audio Source/Destination DLL Sample showing how to write an audio source/destination.

 
A clock you can talk to Talking clock.

 
Basic answering machine Visual Basic answering machine.

 
Basic Voice Commands Basic applet demonstrates Voice Commands.

 
Big Mouth Simple Visual Basic sample for TTS with a big mouth.

 
C++ Wrapper Demo Use the C++ wrapper to make application development easier.

 
Colors Java memory game using speech recognition.

 
Demo Dictation Pad An application you can dictate into.

 
Dictate using the direct speech API An application you can dictate into.

 
Dictation Pad in Visual Basic Visual Basic version of Dictation Pad.

 
DirectSound Audio Destination This application has three text-to-speech voices talking at the same time.

 
E-Mail Notification Alerts you when new mail arrives into Microsoft Outlook (Office version).

 
Fake Speech Recognition and Text-to-Speech Engines The fake speech recognition and text-to-speech engines can be used as templates for engine vendors wishing to port their existing technology to SAPI.

 
Feedback Application Demonstrate how to write a UI feedback application for Voice Commands.

 
Full Duplex Demo Allows you to do speech recognition and text-to-speech at the same time on full duplex sound cards.

 
IE Wrapper Object The Internet Explorer wrapper object is necessary for engine vendors to produce self-extracting executables that are automatically downloaded from web pages.

 
Lexicon Editor Allows you to edit the lexicon of a speech recognition or text-to-speech engine.

 
Make your own TTS voices Tools to make a concatenated wave text-to-speech voice using your own recordings.

 
Microphone Setup Wizard Demo how to use the microphone setup wizard.

 
Microsoft Dictation An application you can dictate into.

 
Microsoft Excel Visual Basic script in Microsoft Excel that allows you to speak in numbers and have them read back.

 
Microsoft Outlook Secretary Access email, contacts, appointments and other Microsoft Outlook features over the phone.

 
Microsoft Mouth A talking mouth.

 
Microsoft Voice Use Voice Commands to control PC applications.

 
Microsoft Word, using voice dictation Visual Basic script in Microsoft Word that allows you to dictate directly into Word.

 
Movie Information Sample Sample movie answering service using telephony controls.

 
Phone Converter Demo Demonstrate converting phoneme strings to/from IPA.

 
Pizza Ordering Sample Sample pizza ordering service using telephony controls.

 
Record/Play Records audio from Voice Commands and Voice Dictation applications. It also plays audio into Voice Commands and Voice Dictation applications.

 
Shareware Registration Application Register shareware application over the phone.

 
Speak what you type Simple text-to-speech demo.

 
Speak text from the command line Command line utility which speaks the command line or stdin.

 
Speech Recognition Functionality Test Tests how compatible the engine is with SAPI.

 
Speech Recognition Grammar Compiler Grammar compiler.

 
Speech Recognition Hand Tester Use this to try all the speech recognition APIs.

 
Speech Recognition Load Tester Helps you determine how much memory and CPU the speech recognition engine requires under different system configurations.

 
Speech Recognition Stress Test Tests the stability of a speech recognition engine under stressful conditions, such as long usage, multiple instances, or low resources.

 
Speech Wrapper Library Source code for the C++ speech wrapper.

 
TAPI Test Application Check to see if your TAPI driver/card supports enough functionality to run the telephony demos.

 
Telephony Control Editor Load, edit, and test telephony controls.

 
Telephony Control Sample Sample telephony control code.

 
Telephony Log Viewer View output logs from your telephony application.

 
Text-to-Speech Functionality Test Tests how compatible the engine is with SAPI.

 
Text-to-Speech Hand Tester Try all the direct text-to-speech APIs.

 
Text-to-Speech Load Tester Helps you determine how much memory and CPU a text-to-speech engine requires under different system configurations.

 
Text-to-Speech Stress Test Tests the stability of a text-to-speech engine under stressful conditions, such as long usage, multiple instances, or low resources.

 
Text-to-Speech Queue Sample Queues up male and female voices, and wave files.

 
Try out some voice commands Demo of Voice Commands.

 
Virtual Burger Sample Text-To-Speech and Voice recognition to order a hamburger.

 
Voice Text Hand Tester Try out all the voice text APIs.

 
Wave Editor Allows you to edit wave files.

 
Wave List Editor Analyzes telephony logs for prompts used by your application, and allows you to record the prompts or pre-synthesize them.

 
Web Page Using Text-To-Speech Web page with a talking mouth.
