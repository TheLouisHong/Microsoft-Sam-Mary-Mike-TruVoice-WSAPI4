For Telephony

This is a list of applications related to telephony.

 

Samples new to 4.0 are highlighted in yellow.

 

Accuracy Test Allows you to test the accuracy of your speech recognizer and grammar.

 
Answering Machine Sample Sample answering machine using telephony controls.

 
Basic answering machine Visual basic answering machine.

 
C++ Wrapper Demo Use the C++ wrapper to make application development easier.

 
Make your own TTS voices Tools to make a concatenated wave text-to-speech voice using your own recordings.

 
Movie Information Sample Sample movie answering service using telephony controls.

 
Pizza Ordering Sample Sample pizza ordering service using telephony controls.

 
Shareware Registration Application Register shareware application over the phone.

 
Speech Recognition Grammar Compiler Grammar compiler.

 
Speech Recognition Load Tester Helps you determine how much memory and CPU the speech recognition engine requires under different system configurations.

 
Speech Recognition Stress Test Tests the stability of a speech recognition engine under stressful conditions, such as long usage, multiple instances, or low resources.

 
TAPI Test Application Check to see if your TAPI driver/card supports enough functionality to run the telephony demos.

 
Telephony Control Editor Load, edit, and test telephony controls.

 
Telephony Control Sample Sample telephony control code.

 
Telephony Log Viewer View output logs from your telephony application.

 
Text-to-Speech Load Tester Helps you determine how much memory and CPU a text-to-speech engine requires under different system configurations.

 
Text-to-Speech Stress Test Tests the stability of a text-to-speech engine under stressful conditions, such as long usage, multiple instances, or low resources.

 
Text-to-Speech Queue Sample Queues up male and female voices, and wave files.

 
Wave Editor Allows you to edit wave files.

 
Wave List Editor Analyzes telephony logs for prompts used by your application, and allows you to record the prompts or pre-synthesize them.
