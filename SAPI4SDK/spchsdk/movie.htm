Movie Information Sample

The Movie Information sample connects to a TAPI device (or the PC's speaker and microphone) and demonstrates how to write a telephony application.

 

You need this to run:

  • Command & Control Speech Recognition Engine (Some engines only support one instance, so if you run the application with a multi-line telephony card, only the first line may be active.)
  • Text-to-speech engine.
  • TAPI device. Read this to make sure you have a Voice TAPI driver installed.
