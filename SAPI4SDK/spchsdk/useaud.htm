Use Audio Source/Destination Objects

This is a list of the sample applications demonstrating the use of an audio source/destination object.

 

Samples new to 4.0 are highlighted in yellow.

 

Accuracy Test Allows you to test the accuracy of your speech recognizer and grammar.

 
Audio Destination File Writes the text-to-speech audio to a wave file.

 
Audio Indexer The audio indexer demonstrates how to use a speech recognition engine to generate a keyword index from wave files.

 
Audio Source/Destination DLL Sample showing how to write an audio source/destination.

 
Dictate using the direct speech recognition API An application you can dictate into.

 
DirectSound Audio Destination This application has three text-to-speech voices talking at the same time.

 
Microsoft Mouth A talking mouth.

 
Telephony Control Editor Load, edit, and test telephony controls.
