Direct Speech Recognition Hand Tester

SRTest is an MFC dialog application for testing the speech recognition interfaces, recognition engines, and grammars. It allows comprehensive manual testing of the speech recognition API and speech recognition engine. SRTest uses the following interfaces: ISRAttributes, ISRCentral, ISRDialogs, ISREnum, ISRFind, ISRGramCFG, and ISRGramCommon.

SRTest will allow one speech recognition mode to be loaded, and multiple grammar objects per each loaded mode.

To get SRTest up and working:

  1. Run it
  2. Press "Add Mode"
  3. Under "Add Mode," select the engine you wish to test, and press OK.
  4. In the main dialog, select one of the grammar (.grm) files to use.
  5. If there are no .grm files showing up, press the "Rescan Files" button and type in "c:\program files\Microsoft speech sdk\sr".   (Note: If you installed your SDK to a different directory, you will have to type a different path name.) Press OK, and some grammar files will show up in the list.
  6. If you wish to make your own grammars, use the Speech Recognition Grammar Compiler. Make sure to save the grammars as .grm files instead of .txt or .srg.
  7. Double click a grammar to load it, such as tv.grm. The grammar name will appear in the "Loaded grammars" list in the upper right hand corner.
  8. To use a grammar, double click the name in the "Loaded grammars" list. This will change the tab dialogs to the grammar interfaces. Double clicking also activates the grammar.
  9. Double click tv.grm.
  10. Speak "CNN" or "NBC". You'll see notifications indicating what the speech recognizer heard.
  11. To stop listening, press "Deactivate."

 

The speech recognition hand tester provides a lot more functionality than just this. In general, there's a tab/pane for every interface, and each tab/pane has a button or edit field for each function in the interface. This enables you to access features of the speech recognition engine by hand and test out its responses.

 

Troubleshooting

At startup, SRTest scans the grammar directory for available grammar files (*.grm). The grammar directory is retrieved from a registry variable (HKEY_CURRENT_USER\software\Microsoft\srtest\settings) which is added to your registry by setup. If no grammar files are found, it brings up a dialog asking for a path. Type the full path name of the directory where the grammar files are. To change the path at a later time press the Rescan Grammar Files button in the ISRCentral property page. If grammar files are found in the new location, that path will be added to the registry.

Notes

There are no 'Apply' buttons for edit controls. To apply a value which you have just entered press TAB.
