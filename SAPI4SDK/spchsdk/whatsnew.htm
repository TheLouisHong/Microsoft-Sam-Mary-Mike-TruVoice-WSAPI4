What's new in 4.0

  • Lots of telephony
    • New objects designed for telephony
    • New objects designed for using speech and TAPI
    • Telephony controls
    • Telephony sample applications
    • Telephony tools
  • ActiveX components
    • Use in Visual Basic, VB-Script, Visual Basic for Applications, Java, Java-Script
    • Lots of samples
  • Continuous dictation support in Voice Dictation
  • Other samples and tools
    • Grammar Compiler - Easy to use grammar compiler
    • Wave editor
    • Animated mouth
    • Speech recognition and text-to-speech stress tests
  • New direct text-to-speech and speech recognition APIs

 

What changed from 2.0 to 3.0

  • New APIs:
    • A new high level module, "Voice Dictation" that allows applications to easily incorporate dictation.
    • Several new audio source/destinations: Recognize from a wave file, text-to-speech to a wave file, monitor the audio going to the recognizer, and send text-to-speech to DirectSound
    • Speech toolkit APIs: Microphone Setup Wizard, Phoneme Conversion Module
  • "Local web site" that allows you to quickly and easily find what you're looking for in the SDK.
  • Documentation loadable by VC 5.0.
  • Lots more demo applications and sample source code. Including:
    • Text-to-Speech demo application with lip syncronization
    • Dictation demos and sample source code.
    • An accuracy test application.
    • Simple speech driven adventure game
    • More sample audio source and destination objects
