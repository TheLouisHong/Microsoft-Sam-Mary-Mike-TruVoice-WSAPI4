Audio Source/Destination Sample DLL

AUDIOSD.DLL contains sample code for an audio source and destination object that record/play from/to the multimedia devices installed on the Windows system. It is very similar to the code shipped by Microsoft for the CLSID_MMAudioSource and CLSID_MMAudioDest.

To install the object into OLE, edit AUDIOSD.REG so that the listed path-names are pointing to the location of AUDIOSD.DLL on your PC. Double-click the registry file in the explorer or file-manager to install the registry entries.

To use the object from your software, replace references to CLSID_MMAudioSource or CLSID_MMAudioDest with CLSID_MyAudioSource or CLSID_MyAudioDest.

That's it. The speech recognition or text-to-speech engines will then be using the custom audio object.

You can debug the code from Microsoft Visual C++ by opening MSVC with the AUDIOSD.MAK project. Put a breakpoint at DllGetClassObject(). Modify the EXE to run in Project/Settings/Debug to your application. Run Debug\Go. When your object is created, MSVC will stop at the breakpoint.
