Dictate using the direct speech recognition API

The “Dictation Demo” program demonstrates the direct speech recognition API, allowing you to load in a dictation or limited-domain grammar. You can leave the grammar field blank to do general purpose dictation. Make sure to select a dictation engine in the engine drop-down.

To run the applications:

  1. Run it
  2. You can leave the grammar file name blank.
  3. Make sure the speech recognition engine in the "Speech engine to use" drop-down is OK.
  4. Press OK.
  5. Dictate some text, such as "This is a test .\period". The application does not show hypothesis, so the text will not appear until the speech recognizer has transcribed the sentence, which may take a few seconds.

DirectSound Capture Audio Source- VDctDemo.exe demonstrates the use of DirectSound Capture, a way of recording using Direct Sound 5.0. To use the DirectSound Capture object, modify the source code so DIRECTSOUND is defined in both .cpp files. This will only work if DirectSound 5.0 is installed on your system.
