Microsoft Word Embedded Dictation Sample

You can dictate into Microsoft Word. Copy the DICTATION.DOT file to your \program files\Microsoft office\templates directory. To create a new document for dictation, use the New.. item on Microsoft Word's file menu and select the dictation.DOT template.

  • Microsoft Word Dictation template - Embeds a dictation object into a Microsoft Word document. This allows dictation with the correction window.
